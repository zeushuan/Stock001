## 🎯 實務操作 SOP

### 每天開盤前
1. 打開 tv_app（雲端 / 本機）
2. 看「📊 今日 TOP 200 即時掃描」
3. 進場欄看：
   - T1 N D 進場（金叉新鮮）
   - T3 拉回進場 ⬤⬤⬤⬤⚪（信心度 4+）
   - T4 N D 反彈（罕見但精準）
4. 點「💾 存」一鍵存成自選股清單

### 進場執行
1. 盤中等待價格回到當日 VWAP
2. 在 ≤ VWAP 掛限價買單
3. 每檔等權配資（建議 1/N % 資金）

### 持倉管理
1. 設停損價：當日收盤 - ATR × 2.5
2. 加碼條件：累積獲利 ≥ 5% + 新訊號（P5）
3. 出場條件：RSI > 70 / EMA20 死叉 / 觸停損

### 出場執行
1. 觸發出場訊號後
2. 盤中等價格回到當日 VWAP 之上
3. 在 ≥ VWAP 掛限價賣單
4. 觸停損則市價立刻出（不等 VWAP）
