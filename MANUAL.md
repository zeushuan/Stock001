
## ① 市場環境

判斷大方向：多頭 / 空頭 / 假多頭

### 顯示範例
```
✅ 多頭市場  黃金交叉 6 天前 ⭐ Sweet Spot ｜ EMA20 > EMA60 ｜ ADX 32.0 ≥ 22（🇹🇼 TW 趨勢有效）
```

### 拆解

| 項目 | 含義 |
|------|------|
| **多頭/空頭** | EMA20 > EMA60 = 多頭 / EMA20 < EMA60 = 空頭 |
| **黃金交叉 N 天前** | EMA20 上次穿越 EMA60 距今 N 個交易日 |
| **EMA20 / EMA60** | 20 日 / 60 日指數移動平均 |
| **ADX** | 趨勢強度指標 14 日週期 |
| **趨勢門檻** | TW: ADX≥22 / US: ADX≥18 |

### 三種狀態

| 狀態 | 含義 | 操作 |
|------|------|------|
| ✅ **多頭市場** | EMA20>EMA60 + ADX≥門檻 | 可考慮進場 |
| ⚠️ **假多頭警告** | EMA20>EMA60 但 ADX<門檻 | 不操作（趨勢弱）|
| 🚫 **空頭市場** | EMA20<EMA60 | 只看 T4 反彈訊號 |

### 黃金交叉天數標記（v9.10k 新增）

依市場研究結果顯示「最佳進場時機」：

**🇹🇼 台股**
- 🌱 **Day 1-4** 偏早（等趨勢確認）
- ⭐ **Day 5-7 Sweet Spot**（研究 RR 0.051 最高）
- 仍可進場 **Day 8-15**

**🇺🇸 美股 / 加密**
- ⚡ **Day 1-5 早鳥期**（研究 RR 0.035 最高）
- ⚠️ **Day 6-10** 已過早鳥（衰減 -17%）

## ② 進場判斷

四種進場觸發類型：

### T1 黃金交叉（最積極）
```
✅ T1 黃金交叉  6 天前 ← 積極進場
```
- 條件：**多頭 + ADX 達標 + 黃金交叉 1-10 天內**
- 含義：新多頭啟動，趨勢剛確認
- 操作：可立即進場

### T3 多頭拉回（最常見）
```
✅ T3 多頭拉回  RSI 47.7 < 50 拉回到位 ← 可進場
```
- 條件：**多頭 + ADX 達標 + RSI < 50**
- 含義：強勢股短期回檔，是低吸機會
- 操作：可進場（RSI 越低越好）

### T4 空頭反彈（罕見）
```
🟡 T4 空頭反彈  RSI 28.3 < 32 且連 2 日上升
```
- 條件：**空頭 + RSI < 32 + 連續 2 日上升**
- 含義：超賣後反彈訊號
- 操作：嚴格停損 ATR×2.0 進場

### 飆股模式
```
🔥 飆股 (T1 + ADX≥30)
```
- 條件：T1 黃金交叉 + ADX≥30
- 含義：強趨勢新啟動
- 操作：持到 EMA 死叉才出（不用 RSI）

### T3 信心度 5 指標（v9.9t 新增）

```
📊 T3 信心度  ●●●●○ 4/5  高信心 ✨
   ✅ close > EMA20
   ✅ EMA20 5 日上升
   ✅ EMA5 5 日上升
   ✅ EMA5 > EMA20（多頭排列）
   ☐ EMA5 + EMA20 都上升
```

| 分數 | 含義 | 動作建議 |
|------|------|---------|
| ●●●●○ 4-5/5 | 高信心 ✨ | 加重部位 |
| ●●●○○ 2-3/5 | 中信心 | 標準部位 |
| ●○○○○ 0-1/5 | 低信心 ⚠️ | 縮減部位 |

研究：高信心 6 年 RR 0.059 vs 低信心 0.039（**+50%**）

## ③ 出場停損

ATR 動態停損價格

### 顯示範例
```
🛡 停損價 85.53  （收盤 111.27 − ATR×2.5 25.74 = -23.1%）
```

### 計算邏輯

```
停損價 = 收盤 - ATR × 倍數

ATR 倍數規則：
  一般股：     ×2.5
  ADX≥30 飆股：×3.0（給更寬空間）
  反向 ETF：   ×1.5（嚴格）
  T4 反彈：    ×2.0（嚴格）
```

### 跌幅%

跌幅 = -ATR×倍數 / 收盤 × 100

例：111.27 - (5.05 × 2.5) / 111.27 = -11.3%

跌幅越大 = 風險越大 / 但給趨勢空間

## ④ 出場獲利

預估「該何時離場」+ 觸發條件

### 安全持倉狀態
```
✅ 安全持倉
🛡 穩健股模式（ATR/P 2.2% ≤ 3.5%）
🛡 EMA死亡交叉：✅ 差距 11.6%（趨勢持續）
🛡 ADX 32.0 ≥ 25 → 強趨勢，持到死叉
```

### 三種出場觸發

#### 1. EMA20 死叉 EMA60（最重要）
```
🛡 EMA死亡交叉：⚠️ 差距 0.4%（即將死叉！準備出場）
```
- **死叉 = 多頭結束** → 立刻出場
- 「即將死叉」(< 1 ATR) → 部位減半 + 上移停損

#### 2. RSI 出場（穩健股）
```
📤 ADX 23.5 < 25 → RSI出場（門檻75）：RSI 45.2，距出場門檻 75 還差 29.8 點
```
- 條件：ADX < 25 + RSI > 75
- 含義：穩健股過熱該獲利了結

#### 3. ATR 停損
```
🛡 穩健股：停損 ATR×2.5（ADX≥30 用 ×3.0）
```
- 觸停損價 → 強制出場（風控）

### 飆股模式（不用 RSI 出場）

```
🚀 飆股模式（ATR/P 9.3% > 3.5%）
🛡 EMA死亡交叉：差距 11.6%（趨勢持續）
🛡 RSI出場：停用（飆股模式：RSI出場砍掉主升段，回測損失 +400%）
🚀 持倉到EMA死叉為止，不提前出場
```

研究：飆股用 RSI 出場會砍掉主升段，回測損失 +400%。改用「持到死叉」策略。

## ⑤ 推薦策略

系統根據環境 + 進場 + 風險的綜合建議

### 五種推薦類別

#### ⑦ 自適應 T3（拉回進場）
```
⑦ 自適應T3（拉回進場）
進場：立即進場（RSI 44.7 < 50, T3 拉回進場）
出場：ADX < 25 時 RSI > 70 出場；ADX ≥ 25 持到死叉
停損：ATR × 2.5（ADX≥30 用 ×3.0）
```

#### ② 趨勢 EMA（飆股模式）
```
② 趨勢EMA（飆股模式）
進場：立即進場（T1 黃金交叉），不等拉回
出場：持到 EMA 死亡交叉才出場
停損：ATR × 2.5 作底線停損，獲利 ≥30% 後改用死叉追蹤
```

#### ⛔ 不進場 — 即將死叉（v9.10j）

當「進場條件達成」+「EMA20-EMA60 < 1 ATR」時觸發：
```
⛔ 不進場 — 即將死叉
原訊號 T1/T3 雖達成，但 EMA20 距 EMA60 僅 X 倍 ATR，
隨時可能死叉，建議觀望避免「進場後立刻出場」陷阱
```

#### ⚠️ 不建議進場（保守風格 vs 高風險）

當保守風格（IND+DXY）遇到高波動股 (ATR/P>5%)：
```
⚠️ 不建議進場（保守風格 vs 高風險訊號）
建議：①縮減部位至 1/2 ②手動上移停損至 EMA60 下方 ③改選平衡/進攻風格
```
**注**：⭐ TW 最佳 / ⭐ US 最佳 不會觸發此警告（v9.10o 修復）

## 估值參考

EPS / PER / PBR / 殖利率

### 顯示範例
```
💰 估值參考  EPS(TTM) 10.81 ｜ PER 32.1 (偏高（成長股）) ｜ PBR 8.79 ｜ 殖利率 —
```

### PER（本益比）顏色判定

| PER | 顏色 | 含義 |
|-----|------|------|
| < 0 或 > 100 | 🔴 紅 | 虧損或極端高估 |
| 10-20 | 🟢 綠 | 合理偏低 |
| 20-30 | 🟡 黃 | 合理 |
| 30-50 | 🟠 橙 | 偏高（成長股）|
| > 50 | 🔴 紅 | 過熱 |

### PBR（股價淨值比）

```
PBR 1.0 = 股價 = 每股淨值
PBR < 1.0 = 跌破淨值（價值低估）
PBR > 5.0 = 高估（除非高 ROE）
```

### 殖利率

```
殖利率 = 年配息 / 股價 × 100
> 5%  高殖利率（穩健投資）
3-5% 一般
< 3%  低（成長股）
—     未配息或無資料
```

## 常見問答

### Q1: 為什麼進場條件達成卻顯示「不建議進場」？

**A**: 三個可能原因：
1. **即將死叉**（EMA20-EMA60 < 1 ATR）→ 觀望避免進場後立即出場
2. **保守風格 vs 高波動**（ATR/P > 5%）→ 部位太大風險高
3. **過度延伸**（距 SMA200 > 40%）→ 追高風險

### Q2: 為什麼台股顯示 ADX 22 / 美股顯示 ADX 18？

**A**: 美股研究發現 P10+POS+ADX18（寬鬆趨勢）效果最佳（RR 0.496），台股維持 ADX22 預設（P5+VWAPEXEC RR 0.611）。

### Q3: 飆股模式跟一般股有什麼差別？

**A**: 飆股 (ADX≥30 + ATR/P>3.5%)：
- 停損 ATR×3.0（更寬）
- **不用 RSI 出場**（飆股 RSI>70 還會繼續飆）
- 持到 EMA 死叉為止
- 一般股反之：ATR×2.5 + RSI>70/75 出場

### Q4: 「黃金交叉 6 天前 Sweet Spot」是什麼意思？

**A**: 研究發現：
- 🇹🇼 台股 Day 5-7 平均 RR 0.051 最高（vs Day 1 RR 0.038）
- 🇺🇸 美股 Day 1-5 早鳥期 RR 0.035 最高（之後衰減）
- 黃金交叉當天進場容易碰假突破

### Q5: T3 信心度 4-5/5 vs 0-1/5 差多少？

**A**: 全市場 6 年回測：
- 高信心 (4-5/5)：RR 0.059
- 低信心 (0-1/5)：RR 0.039
- **差 +50%** — 高信心建議加重部位

### Q6: 為什麼有些股票顯示 +0% 或 -10%？

**A**: 個股 row 末端的色塊 = **過去 60 日價格動量**：
- 🟢 綠 ≥+5%：強勢
- 🔵 藍 0-5%：穩定
- 🟠 橙 -10~0%：下行
- 🔴 紅 <-10%：弱勢

### Q7: 「跌深反彈訊號」對台股也有效嗎？

**A**: **無效甚至反向**！跨市場研究發現：
- 🇺🇸 US 跌深反彈 RR +0.146（最強訊號）
- 🇹🇼 TW 跌深反彈 RR -0.000（無效）
- 兩市場「逆勢 vs 順勢」邏輯完全相反

### Q8: VWAP 進場/出場建議只對台股顯示？

**A**: 對。VWAPEXEC 需要 5-min bar 資料（玉山 Fugle API），美股無此資料。

### Q9: TOP 200 訊號顯示「過期 N 天」怎麼辦？

**A**: 點右側「🔄 雲端更新」按鈕（~10 秒）：
- TW: 用 yfinance 抓最新一日，更新 top200_signals.json
- US: 同上
- 雲端不需 data_cache，直接抓 yfinance

### Q10: 個股名稱顯示亂碼或英文 ticker？

**A**: 雲端版用 `us_full_tickers.json` 的 detail 對應名稱：
- TW: 從 tw_stock_list.json 載入
- US: 從 us_full_tickers detail 載入（簡化處理）
- 若仍顯示 ticker = name_map 沒對應到該股
