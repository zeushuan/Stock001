## 為什麼沒有突破 1.03？

### 已驗證的訊息上限證明

```
ML 模型上限：DecisionTree CV 75.6%
  └─ 主要靠 v7_pct (37%) + bh_pct (15%)（事後特徵）
  └─ 即時可用特徵（bull_days, RSI 等）已被各種旗標測過

RL 模型結果：學到 POS 規則 + RSI 高位避險 + 8+ 倉位避險
  └─ 與人工規則高度一致
  └─ 證明當前訊息集合已被人工充分利用
```

### 真正能突破的方向（需要新資料源）

1. **真實財報日** - 公司級別精準避險
2. **盤中分鐘級資料** - 開盤跳空、收盤前波動
3. **法人籌碼** - 三大法人買賣超
4. **基本面整合** - ROE、P/E、營收年增率
5. **產業相對強弱** - 同業比較
6. **新聞情感分析** - NLP 處理新聞
7. **跨股票協同** - Pairs trading（被排除）
